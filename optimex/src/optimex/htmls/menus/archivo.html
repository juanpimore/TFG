Nueva clase

Genera una plantilla de clase java para comenzar a programar una clase nueva. Si anteriormente había una clase abierta a la que se le habían realizado cambios, se muestra un diálogo que nos pregunta si queremos guardar esos cambios antes de abrir la clase nueva.


Guardar clase

Guarda la clase abierta en el editor. Si la clase es nueva y no existe en un archivo, se muestra un diálogo que nos permite seleccionar el nombre del archivo y el lugar donde guardarlo.


Cargar y compilar clase

Muestra un diálogo donde se puede seleccionar un fichero java para abrir en el editor y se compila inmediatamente después. Si anteriormente había una clase abierta a la que se le habían realizado cambios, se muestra un diálogo que nos pregunta si queremos guardar esos cambios antes de abrir la clase nueva.


Compilar clase

Se compila la clase abierta en el editor. A continuación se muestra el resultado de la compilación.


Introducir datos por teclado

Muestra un diálogo en el que podemos introducir cada uno de los valores de los que está compuesto un dato.


Generar datos aleatorios

Muestra un diálogo pidiendo el número de datos que se quieren generar, una vez introducido el número, se muestra otro diálogo en el que introducir los rangos de aleatoriedad de los datos a generar.La aplicación generará los datos solicitados.


Cargar datos de fichero

Muestra un diálogo donde se puede seleccionar un fichero xml con los datos necesarios para la ejecución de los métodos selecionados. Este fichero se abre y se guardan los datos en la tabla de datos de la aplicación.


Exportar tabla datos

Guarda los datos que se estan utilizando actualmente para la ejecución. Muestra un diálogo donde se puede seleccionar o crear un fichero en el que guardar los datos.


Exportar tabla resultado

Muestra un diálogo donde se puede seleccionar o crear un fichero en el que guardar los resultados de una ejecución.


Exportar tabla histórica

Muestra un diálogo donde se puede seleccionar o crear un fichero en el que guardar los resultados de todas las ejecuciones mostrados en la tabla histórica.


Exportar tabla resumida

Muestra un diálogo donde se puede seleccionar o crear un fichero en el que guardar los porcentajes de optimidad de cada método después de las ejecuciones realizadas.


Exportar todas las tablas

Muestra un diálogo donde se puede seleccionar el fichero excel en el que guardar todas las tablas generadas después de las ejecuciones realizadas.


Salir

Cierra la aplicación.
