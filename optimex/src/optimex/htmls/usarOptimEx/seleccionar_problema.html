Seleccionar problema

Seleccione Ejecutar > Seleccionar problema desde la barra de menus.



A continuación aparecerá el selector de signatura del problema y el selector de objetivo del problema, en el que se elige entre Maximizar y Minimizar:



Maximizar

Con esta opción se marcarán en gris el resultado máximos de cada ejecución en la Tabla histórica.


Minimizar

Con esta opción se marcarán en gris el resultado mínimos de cada ejecución en la Tabla histórica.
