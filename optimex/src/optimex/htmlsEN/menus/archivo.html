File

New class

Generates a java class template to start a new class schedule. If you previously had an open class to which they had made changes, shows a dialog that asks if you want to save these changes before opening the new class.


Save class

Save the open class in the editor. If the class is new and does not exist in a file, it shows a dialog that lets you select the file name and where to save it.


Load and compile class

Displays a dialog where you can select a file to open in java editor and compiled immediately. If previously open class to which it had made changes, it displays a dialog which asks us if we want to save those changes before opening the new class.


Compile class

It compiles the open class in the editor. Below is the result of compilation.


Enter data from keyboard

Displays a dialog where you can enter each of the values is a given compound.


Generate aleatory data

Displays a dialog asking for the number of data to be generated once entered the number, it displays another dialog in which to enter the ranks randomness of data generated. The application will generate the requested data.


Load data from file

Displays a dialog where you can select an xml file with the data needed to executing selecionados methods. This file is opened and saved data in the data table of the application.


Export data table

Save the data you are currently using for execution. Displays a dialog where you can select or create a xml file in which to save the data.


Export result table

Displays a dialog where you can select or create a file in which to save the results of execution.


Export historical table

Displays a dialog where you can select or create a file in which to save the results of all executions shown in the historical table.


Export summarized table

Displays a dialog where you can select or create a file in which to save the percentages of optimality of each method after the executions.


Export all the tables

Displays a dialog where you can select or create a excel file in which to save all the tables in diferents tabs.


Exit

Closes the application.
