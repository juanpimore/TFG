Editor panel


The editor panel is a simple text editors with syntax coloring, line numbering and folding regions:

In this editor, we find the usual buttons in any text editor:
  1. Undo
  2. Redo
  3. Cut
  4. Copy
  5. Paste
  6. Goto Line ..
  7. Search ..
  8. Find Next
  9. Select All
